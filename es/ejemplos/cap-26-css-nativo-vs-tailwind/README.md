# Ejemplos del capitulo 26 · CSS nativo vs Tailwind

> Etiqueta de soporte: **N/A** (va de una decisión, no de una feature de CSS)

## La decisión honesta, en forma de árbol

El `index.html` es un árbol de decisión que puedes seguir para elegir entre CSS
nativo moderno y Tailwind, con tres preguntas sobre tu equipo, tu proyecto y tu
momento. Los resultados van codificados por color: CSS nativo, seguir con Tailwind,
o cualquiera.

Es HTML y CSS puro, sin build. La idea es el criterio, no la respuesta: elige por el
contexto, no por el bando, y revísalo cada par de años.

**Ábrela:** el `index.html` de esta carpeta, o en vivo en
https://alexanderuk82.github.io/modern-css-ebook/es/c26/decision/
