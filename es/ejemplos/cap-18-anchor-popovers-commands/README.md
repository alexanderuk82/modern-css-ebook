# Ejemplos del capitulo 18 · Anchor, popovers e invoker commands

> Etiqueta de soporte: **mixta** · Popover API BASELINE 2024/2025; anchor positioning y commandfor CHROME ONLY · más nuevo: interestfor

## Tooltips y menús que se colocan solos, sin Floating UI

Dos demos en una página:

- **Badge anclado a una esquina**: un badge de notificación clavado a la esquina superior derecha de la tarjeta con `anchor()`. Sin medir, sin JavaScript.
- **Popover, cero JavaScript**: un botón abre un popover con la Popover API (`popovertarget`), anclado debajo con `position-area` y que se voltea arriba con `position-try-fallbacks` si no cabe.

**Ábrela:** el `index.html` de esta carpeta, o en vivo en
https://alexanderuk82.github.io/modern-css-ebook/es/c18/anchor/

`commandfor`/`command` e `interestfor` se cubren en el capítulo con sus etiquetas.
