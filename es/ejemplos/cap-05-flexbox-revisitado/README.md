# Ejemplos del capitulo 5 · Flexbox revisitado

> Etiqueta de soporte: **BASELINE**

## Una dimensión, bien hecha

Dos demos en una página:

- **Nav responsive**: una barra flex con `flex-wrap` y `gap`, con el botón de acción empujado al borde por `margin-inline-start: auto`. Se envuelve sola, sin media queries.
- **Barra lateral que se apila**: `flex: 1 1 14rem` para la barra y `flex: 999 1 20rem` para el contenido. Una al lado de otra cuando hay sitio, apiladas cuando es estrecho.

**Ábrela:** el `index.html` de esta carpeta, o en vivo en
https://alexanderuk82.github.io/modern-css-ebook/es/c05/flex/

Todo esto es BASELINE y funciona en todos los navegadores hoy.
