# Chapter 4 examples · Grid in 2026

> Support label: **BASELINE**

## Whole layouts with very little code

Two demos in one page:

- **Dashboard from named areas**: a header, nav, main and footer laid out with `grid-template-areas`. The CSS is drawn with words, and it holds as you resize.
- **auto-fit gallery**: a card grid with `repeat(auto-fit, minmax(...))` that reflows from many columns to one with no media query.

**Open it:** `index.html` in this folder, or live at
https://alexanderuk82.github.io/modern-css-ebook/en/c04/grid/

Everything here is BASELINE and works in every browser today. Subgrid and
masonry get their own chapter (7).
