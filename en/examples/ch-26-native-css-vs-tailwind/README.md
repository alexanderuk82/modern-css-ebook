## The honest decision, as a tree

`index.html` is a decision tree you can follow to choose between native modern CSS
and Tailwind, based on three questions about your team, project and moment.
Outcomes are colour coded: native CSS, keep Tailwind, or either.

It is plain HTML and CSS, no build. The point is criteria, not an answer: choose by
context, not by side, and revisit every couple of years.

**Open it:** `index.html` in this folder, or live at
https://alexanderuk82.github.io/modern-css-ebook/en/c26/decision/
