# Chapter 5 examples · Flexbox Revisited

> Support label: **BASELINE**

## One dimension, done right

Two demos in one page:

- **Responsive nav**: a flex bar with `flex-wrap` and `gap`, with the CTA pushed to the edge by `margin-inline-start: auto`. It wraps on its own, no media query.
- **Sidebar that stacks**: `flex: 1 1 14rem` for the aside and `flex: 999 1 20rem` for the content. Side by side when there is room, stacked when narrow.

**Open it:** `index.html` in this folder, or live at
https://alexanderuk82.github.io/modern-css-ebook/en/c05/flex/

Everything here is BASELINE and works in every browser today.
