## The same button, four ways

`index.html` shows the same button built four ways, all rendering the same:
classic CSS, BEM, CSS modules, and Tailwind. The Tailwind button uses **real,
compiled Tailwind v4.3 output** in `tw.css`.

The `tw.css` here is generated by `@tailwindcss/cli@4.3.1` from
[`/tailwind-build`](../../../tailwind-build/), scanning this HTML and emitting
only the utilities it uses (`px-4 py-2 rounded-lg bg-violet-700 text-white
font-semibold`). To rebuild it:

```bash
cd tailwind-build
npm install
npx tailwindcss -i input.css -o ../live-demos/en/c24/tailwind/tw.css --minify
```

**Open it:** `index.html` in this folder, or live at
https://alexanderuk82.github.io/modern-css-ebook/en/c24/tailwind/
