# Chapter 13 examples · Modern Typography

> Support label: **BASELINE 2024/2025** (text-box-trim is the newest, BASELINE 2025)

## Professional type without the old tricks

Three demos in one page:

- **text-wrap: balance**: the same heading with and without balance, side by side. Balance evens the line lengths.
- **Fluid type with clamp()**: a heading sized with `clamp(1.4rem, 9cqi, 3.4rem)`. Resize to see it scale.
- **Tabular numbers**: `font-variant-numeric: tabular-nums` lines digits into columns.

**Open it:** `index.html` in this folder, or live at
https://alexanderuk82.github.io/modern-css-ebook/en/c13/type/

`text-box-trim` (optical centering) and `text-wrap: pretty` are covered in the
chapter. text-box-trim is BASELINE 2025, use it with a base padding fallback.
